# ClinicalTrials.gov API — local reference snapshots

These files are local snapshots of the ClinicalTrials.gov API documentation captured on 2025-12-25. They are kept here for reference while GregoryAI adapts to the v2 API.

The canonical and authoritative source is the official documentation:

- API overview: <https://clinicaltrials.gov/data-api/api>
- Search areas: <https://clinicaltrials.gov/data-api/about-api/search-areas>
- Study data structure: <https://clinicaltrials.gov/data-api/about-api/study-data-structure>
- Migration guide: <https://clinicaltrials.gov/data-api/about-api/api-migration>

> Note: The ClinicalTrials.gov data pipeline was modernised on August 26, 2025. Some markup fields and location/geopoint data changed format. These snapshots predate that change. Check the canonical docs for the current state.
